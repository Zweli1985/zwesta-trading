Call: POST /request-activation│\n│ Parameters: botId            │\n└─────────────────────────────┘\n          ↓\n    [Backend Actions]\n    - Generate 6-digit PIN\n    - Store in database\n    - Email PIN to user\n    - Log PIN (for testing)\n          ↓\n┌─────────────────────────────┐\n│ Response: PIN sent          │\n│ Expires: 10 minutes         │\n│ Status: 200 OK              │\n└─────────────────────────────┘\n          ↓\n[User receives PIN in email]\n          ↓\n[User enters PIN in app]\n          ↓\n┌──────────────────────────────────────┐\n│ Call: POST /api/bot/start            │\n│ Parameters:                          │\n│  - botId: \"bot_abc123\"               │\n│  - activation_pin: \"482931\"  ⭐      │\n└──────────────────────────────────────┘\n          ↓\n    [Backend Actions]\n    - Verify PIN exists\n    - Check expiration\n    - Check failed attempts\n    - Delete PIN (one-time use)\n    - Enable bot trading\n          ↓\n      ✅ Bot Started! ✅\n ↓\n┌─────────────────────────────────┐\n│ Call: POST /request-deletion     │\n│ Parameters: botId                │\n└─────────────────────────────────┘\n          ↓\n    [Backend Actions]\n    - ↓\n┌────────────────────────────────────────────┐\n│ Frontend: Final Confirmation Dialog        │\n│ - Warn: \"This cannot be undone\"           │\n│ - Require: User type \"DELETE\"              │\n│   ↓\n    [Backend Actions]\n    - Verify token exists\n    - Check expiration\n    - Check user ownership\n    - Delete from database\n    - Delete from active_bots\n    - Clear all related data\n Authentication\n```\n┌─────────────────┐\n│   User Input    │\n└────────┬────────┘\n         ↓\n┌──────────────────────────────┐\n│ Session Validation           │\n│ @require_session decorator   │\n│ Checks: user_id in session   │\n└────────┬─────────────────────┘\n         ↓\n    ✅ User Authenticated\n```\n\n### Layer 2: Bot Ownership\n```\n    ✅ User Authenticated\n         ↓\n┌──────────────────────────────┐\n│ Bot Ownership Check          │\n│ - In-memory: active_bots[id] │\n│ - Database: user_bots table  │\n└────────┬─────────────────────┘\n         ↓\n    ✅ Bot Belongs to User\n```\n\n### Layer 3: Operation-Specific Security\n```\n    ✅ Bot Belongs to User\n         ↓\n    ┌────────┴─────────┐\n    ↓                  ↓\n[ACTIVATION]      [DELETION]\n    ↓                  ↓\nVerify PIN        Verify Token\nExpiry: 10min     Expiry: 5min\nOne-time use      One-time use\n  ↓                  ↓\n✅ Allow Op       ✅ Allow Op\n```\n\n---\n\n## Withdrawal System
